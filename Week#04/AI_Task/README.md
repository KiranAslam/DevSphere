# Week 4 AI Task: Neural Networks Basics

This folder contains a simple neural network implementation built with Python and NumPy.

## What the script does
- Loads the Iris flower dataset
- Splits the data into training and testing sets
- Normalizes input features
- Trains a small neural network with one hidden layer
- Prints predictions and accuracy

## How to run
From the project root, run:

```bash
python Week#04/AI_Task/neural_network.py
```

## Workflow summary
1. Load data
2. Preprocess features
3. Build the model
4. Train with backpropagation
5. Show predictions and results
