# Week 4 ML Task: Classification Models

This folder contains a simple classification model implemented with scikit-learn.

## What it does
- Uses the Breast Cancer dataset
- Splits the data into training and testing sets
- Standardizes the feature values
- Trains a logistic regression classifier
- Prints accuracy and sample predictions

## How to run
From the workspace root, run:

```bash
python "Week#04/ML Task/classification_model.py"
```
